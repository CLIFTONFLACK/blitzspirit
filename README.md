# BlitzSpirit.co.uk

**T-shirts with a backbone.** British heritage apparel — ironic and iconic designs
evoking the grit of the Blitz. Unbowed. Unbroken. Unforgiving.

## Live site

Production deploys to **https://blitzspirit.vercel.app/** via the linked Vercel
project ([cliftonflacks-projects/blitzspirit](https://vercel.com/cliftonflacks-projects/blitzspirit)).
Pushes to `main` go live.

## Contents

| Path | What |
|---|---|
| `index.html` | Homepage — hero, featured drop, Boudica editorial, heritage band |
| `collection.html` | Shop-all collection grid |
| `product.html` | Product detail page (Boudica) |
| `styles.css` | Design tokens + all component styles |
| `assets/` | Approved product imagery and brand graphics |
| `docs/SHOPIFY_BUILD_SPEC.md` | Spec for translating this design to a Shopify theme |
| `docs/DESIGN_SYSTEM.md` | Brand design tokens (colour, type, components, anti-patterns) |

Static site — no build step. Open `index.html` locally or deploy as-is.

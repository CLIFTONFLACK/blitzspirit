# BlitzSpirit — Lead Generation Playbook

What's already built into the site, and the ideas worth running next. Everything in
the brand voice: defiant, dry, never desperate.

## Built into the site (live)

| Mechanic | Where | Detail |
|---|---|---|
| **"Join the Home Guard" signup modal** | All pages | Fires after 16s or on exit intent. Offers **10% code BACKBONE10**. Suppressed for 7 days once seen, permanently once signed up (localStorage). "No thanks — I'll pay full price" decline button does double duty as a conversion nudge. |
| **Newsletter band** | Every page footer area | "Join the resistance to corrosion" + 10% incentive; inline success state issues the code immediately (instant gratification). |
| **Accessories waitlist** | Homepage tile | "Incoming — join the waitlist" opens the signup modal: pre-launch demand capture for the next category. |
| **Early-access framing** | Help FAQ, Icons archive note | "Subscribers get 48 hours before anyone else" / "files declassified" — positions the list as membership, not marketing. |
| **Story-first inner pages** | story.html, icons.html | Long-form brand narrative builds the emotional case before the ask; every icon file ends in a shop CTA. |

## Next moves (recommended order)

1. **Wire the forms to a real ESP.** Shopify Email or Klaviyo signup endpoint replaces
   the front-end mock; keep the BACKBONE10 auto-issue and the Home Guard framing.
   Tag source (modal / footer / waitlist) for attribution.
2. **Welcome flow ("Enlistment papers")** — 3 emails: (1) code + the Story,
   (2) the Icons files, (3) social proof + bestseller. Dry tone, no exclamation marks.
3. **Drop waitlists per product.** "Declassified soon" product cards with notify-me —
   scarcity that fits the small-run printing story.
4. **"Name the next icon" ballot.** Email-gated vote on the next design file
   (e.g. Nelson, the Land Girls, the Spitfire ground crew). Voters get first purchase
   window. Builds list AND validates designs before printing.
5. **Referral: "Bring a recruit."** Give £5 / get £5, framed as recruitment papers.
6. **Parcel inserts.** Postcard with the icon's "file" on one side, QR to the Home
   Guard + review request on the back. Sticker of the roundel in every order —
   stickers end up on laptops, laptops are billboards.
7. **Quiz: "Which icon are you?"** (Boudica / Roundel / Skull & Brollies / Roll Call)
   — shareable result card, email-gated result. Cheap top-of-funnel for social.
8. **UGC hook: #GoOnThen.** Ask buyers to post the shirt where backbone was required —
   marathon finish lines, hospital discharge days, Monday mornings. Repost weekly
   "Mentioned in Dispatches".
9. **Abandoned basket copy** (when on Shopify): subject "You left your post." Body:
   "Your basket is holding the line alone. 48 hours before we stand it down."
10. **Out-of-stock capture**: back-in-stock alerts on sold-out sizes (Shopify app) —
    highest-intent emails you can collect.

## Measurement

- Modal: view → signup conversion (target 4–8%); decline-button copy A/B test.
- List growth by source tag; first-order conversion of BACKBONE10 within 14 days.
- Waitlist size per upcoming product = print-run sizing signal.
